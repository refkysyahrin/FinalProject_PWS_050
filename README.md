## Halaman Pembuatan API Key
Fitur ini digunakan oleh pelanggan untuk mendapatkan API Key sebagai syarat utama dalam mengakses menu dan melakukan pemesanan.

### Ketentuan:
- Pengguna **tidak dapat mengakses menu** apabila belum memiliki API Key.
- Pelanggan **wajib mendaftar sebagai member baru** untuk mendapatkan API Key.
- Setelah proses pendaftaran berhasil, API Key akan **dibuat secara otomatis** oleh sistem.
- Kolom API Key akan **terisi otomatis** saat pelanggan menekan tombol **OK**.
